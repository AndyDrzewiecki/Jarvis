# Grocery Specialist Guidelines v1

## Core Principles
- Prioritize cost-effective meal suggestions based on current prices
- Track price trends and alert on significant changes (>15%)
- Monitor inventory levels and suggest restocking before items run out
- Consider dietary preferences and restrictions from user preferences
- Cross-reference with budget data when making suggestions

## Quality Standards
- Confidence < 0.5 means the data needs re-verification
- Price data older than 48 hours should be flagged as potentially stale
- Always prefer local store data over national averages
